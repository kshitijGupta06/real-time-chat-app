# Real-Time Chat Application

## Setup Instructions

### Backend:
1. Go to backend folder:
    ```bash
    cd backend
    ```
2. Install dependencies:
    ```bash
    npm install
    ```
3. Start server:
    ```bash
    node server.js
    ```

### Frontend:
1. Go to frontend folder:
    ```bash
    cd frontend
    ```
2. Install dependencies:
    ```bash
    npm install
    ```
3. Start React app:
    ```bash
    npm start
    ```

## Features:
- Real-time messaging
- Chat history display
- Responsive design
